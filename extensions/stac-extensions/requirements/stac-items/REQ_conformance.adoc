[[req_stac-items_conformance]]

[requirement]
====
[%metadata]
identifier:: /req/stac-items/conformance
part:: A server's conformance declaration SHALL be retrieved from the `/conformance` endpoint as defined in the http://docs.ogc.org/is/17-069r3/17-069r3.html#_declaration_of_conformance_classes[Declaration of conformance classes] clause of the http://docs.ogc.org/is/17-069r3/17-069r3.html[OGC API - Features - Part 1: Core] Standard.
part:: An implementation of a server that supports the ability to generate fully valid STAC items SHALL declare the following conformance classes in the conformance declaration:
+
* http://www.opengis.net/spec/ogcapi-records-5/1.0/conf/stac-items
====
